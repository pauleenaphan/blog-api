# Features

## Users:
- Login authentication
- Account creation
- Viewing blog post
- Commenting on post

## Admin:
- Creating post
- Editing post
- Deleting post
- Deleting comments

## Technologies
- REST API Endpoints
- MongoDB
- React.js
- Express.js
- CSS
