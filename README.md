# BlogWog

Blog site where users can view and comment on blog post.

[Live Site](https://google.com)

# Features
## Users:
- Login authentication
- Account creation
- Viewing blog post
- Commenting on post

## Admin:
- Creating post
- Editing post
- Deleting post
- Deleting comments

## Technologies
- REST API Endpoints
- MongoDB
- React.js
- Express.js
- CSS
